
# Blueprint: 통합과학 성취기준 분석 웹사이트

## 1. 개요

통합과학1의 성취기준, 성취수준, 관련 평가 문항 및 탐구활동을 체계적으로 분석하고 시각화하여 교수-학습 설계를 지원하는 반응형 웹 애플리케이션입니다. 사용자는 성취기준을 쉽게 탐색하고, 각 기준에 따른 성취수준별 평가 문항 예시를 확인하며, 주요 교과서의 탐구활동 자료를 비교 분석할 수 있습니다.

## 2. 디자인 및 기능 명세

### 가. 디자인 컨셉: "Modern Art Gallery"
- **미적 목표:** 현대 미술관의 전시처럼 깔끔하고 정제된 느낌을 주며, 정보가 하나의 작품처럼 보이도록 디자인합니다.
- **레이아웃:** 그리드 시스템을 기반으로 한 비대칭 레이아웃으로 동적인 느낌을 줍니다. 충분한 여백을 활용하여 콘텐츠의 가독성을 높입니다.
- **색상 팔레트:** 무채색(흰색, 회색, 검은색)을 기본으로 하며, 포인트 색상으로 차분한 파란색(`oklch(60% 0.15 240)`)과 활기찬 주황색(`oklch(70% 0.2 45)`)을 사용하여 상호작용 요소와 중요한 정보를 강조합니다.
- **타이포그래피:** `Pretendard` 웹 폰트를 사용하여 제목과 본문의 위계를 명확히 하고 가독성을 극대화합니다.
- **시각 효과:** 부드러운 전환(`transition`) 효과와 미묘한 그림자(`box-shadow`)를 사용하여 깊이감과 상호작용의 직관성을 높입니다.

### 나. 핵심 기능
- **성취기준 및 성취수준 탐색:**
  - 통합과학1의 모든 성취기준을 목록 형태로 제공합니다.
  - 각 성취기준은 아코디언 UI로 구현되어, 클릭 시 해당 기준의 성취수준(A, B, C, D, E)이 펼쳐집니다.
  - 성취수준을 클릭하면 하단에 관련 평가 문항이 나타납니다.
- **문항 분석:**
  - 각 성취수준별로 3개의 샘플 문항(합답형 5지선다)을 제공합니다.
  - 문항에는 필요시 이미지(자리표시자 사용)가 포함됩니다.
  - 각 문항 하단에는 해당 문항이 특정 성취기준 및 성취수준과 어떻게 연결되는지에 대한 해설이 제공됩니다.
- **통합 탐구활동:**
  - 주요 출판사별 통합과학 교과서의 탐구활동을 분석하여 목록으로 제시합니다. (샘플 데이터 기반)
- **반응형 웹:** 모바일, 태블릿, 데스크탑 등 모든 화면 크기에서 최적화된 사용자 경험을 제공합니다.

## 3. 현재 작업 계획

1.  **`blueprint.md` 파일 생성:** 프로젝트의 목표와 계획을 문서화합니다. (완료)
2.  **`public/data.js` 파일 생성:**
    - 통합과학1 성취기준, 성취수준(A-E), 문항(3개/수준), 문항 해설, 탐구활동에 대한 고품질 샘플 데이터를 `scienceData` 객체로 구조화하여 저장합니다.
3.  **`public/index.html` 파일 생성:**
    - 시맨틱 태그(`header`, `main`, `section`, `h1`, `nav`)를 사용하여 기본 구조를 설정합니다.
    - `style.css`, `data.js`, `main.js` 파일을 연결합니다.
    - 콘텐츠가 동적으로 채워질 컨테이너 영역(`div#standards-container`, `div#inquiry-container`)을 준비합니다.
4.  **`public/style.css` 파일 생성:**
    - "Modern Art Gallery" 디자인 컨셉에 맞춰 전체적인 스타일을 구현합니다.
    - CSS 변수를 사용하여 색상, 폰트 등 디자인 시스템을 관리합니다.
    - 아코디언 UI, 문항 카드, 버튼 등 상호작용 요소의 스타일을 정의합니다.
    - 미디어 쿼리를 사용하여 반응형 레이아웃을 구현합니다.
