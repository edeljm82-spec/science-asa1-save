💡 API 키란 무엇인가요?

이 시스템은 구글의 최신 인공지능을 사용해 문제를 정밀하게 분석합니다. 인공지능을 무료로 사용하기 위한 일종의 '개인 전용 출입증'이 바로 API 키입니다.

🚀 1분 만에 API 키 발급받는 방법 (초보자용)

  1. 아래 파란색 링크를 클릭해 구글 AI 스튜디오로 이동하세요. (구글 로그인 필요)
    👉 [구글 AI 스튜디오 바로가기]
  2. 화면 왼쪽 위에 있는 파란색 [Get API key] 또는 [Create API key] 버튼을 클릭하세요.
  3. (핵심) '가져온 프로젝트 선택' 칸을 누르고 [+ 프로젝트 만들기]를 선택한 뒤, [키 만들기]를 누르세요.
  4. 'AIzaSy...' 로 시작하는 아주 긴 비밀번호가 생성됩니다. 그 옆의 [복사 아이콘]을 누르세요.
  5. 다시 이 화면으로 돌아와서, 아래 빈칸에 복사한 키를 붙여넣기(Ctrl+V) 하세요!
