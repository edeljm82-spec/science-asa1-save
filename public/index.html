⚙️ 인공지능(AI) API 키 설정

이 시스템은 구글의 최신 인공지능(AI)을 사용해 선생님께서 입력하신 문항을 정밀하게 분석합니다. 인공지능을 사용하기 위한 개인 전용 비밀번호인 'API Key'가 필요합니다. Key는 선생님의 데이터베이스에 안전하게 보관되며, 한 번만 설정하면 이후에는 자동으로 불러옵니다.

🚀 API 키 발급받는 방법 (가져오기)

👉 구글 AI 스튜디오에서 API 키 발급받기 (가져오기)
  1. 위 파란색 버튼을 클릭해 구글 AI 스튜디오로 이동합니다. (구글 로그인 필수)
  2. 화면 왼쪽 위에 있는 파란색 [Create API key] (또는 Get API key) 버튼을 클릭합니다.
  3. 가져온 프로젝트를 선택하거나 [+ Create new project]를 선택한 뒤, [Create API key]를 누릅니다.
  4. 생성된 키('AIzaSy...' 로 시작) 옆의 [복사 아이콘]을 누릅니다.
  5. 다시 이 화면으로 돌아와서, 아래 입력칸에 붙여넣기(Ctrl+V) 하세요!
